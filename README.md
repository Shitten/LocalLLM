# PocketLLM

**Mobile web UI for your local LLM. Use it from your phone. No cloud. No subscription.**

## The Problem

You got LLM running on your PC. Cool. Now try using it from your phone.

- Running LLM on phone? Impractical. Hardware can't handle it.
- Accessing it remotely? You get raw JSON responses. Good luck typing curl commands on a phone.

## The Fix

A clean mobile chat interface that connects to your local LLM. Works over Tailscale. Works from anywhere.

Point it at your Ollama, LM Studio, or llama.cpp endpoint. That's it.
